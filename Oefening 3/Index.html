Definitie lijsten

  • Ongenummerde lijsten
    • Ongenummerde lijsten worden weergegeven met de tag <ul>, elk item met <li>.
  • Genummerde lijsten
    • Genummerde lijsten worden weergegeven met de tag <ol>, elk item met <li>.
  • Opties voor genummerde lijsten
    • Je kunt de attributen style en start meegeven met de <ol> tag.
    • Je kunt het attribuut value meegeven met de <li> tag.
  • Ongenummerde lijsten met subonderwerp
    • Sublijsten krijgen automatisch een ander opsommingsteken
  • Geneste lijsten
    • Je kunt tags <ol> en <ul> combineren.
